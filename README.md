# Escape The Website Challenge

This project is a browser-based puzzle website built using HTML, CSS, and JavaScript.

Users explore the webpage to find hidden clues and solve puzzles in order to unlock the next room and finally escape the website.

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Project Structure
- index.html – Landing page
- room1.html – First puzzle room
- room2.html – Second puzzle room
- final.html – Escape completion screen
- style.css – Styling and UI design
- app.js – Game logic and interactions
